Lottery Edge - About

Lottery Edge is a demonstration web site developed during the University of Washington (UW) Professional & Continuing Education (PCE) class HTML5 Application Programming Interfaces (API). This class is part of the UW certificate program HTML5, CSS3 & Responsive Design For Web Development.

Software sources

  • Bootstrap is a HTML, CSS, and JavaScript framework for developing responsive, mobile first projects on the web.
  • jQuery is a JavaScript library that makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
  • jStat is a statistical library written in JavaScript that allows you to perform advanced statistical operations without the need of a dedicated statistical language (e.g. MATLAB or R).
  • Normalize.css is a CSS file that makes browsers render all elements more consistently and in line with modern standards by precisely targeting only the styles that need normalizing.
